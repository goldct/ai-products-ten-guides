## 第二步：理解套利原则

### 核心原则

✅ **真实信息差** - 确实存在，不是幻觉
✅ **利润 > 成本** - 算清真实利润（含时间、手续费）
✅ **可快速执行** - 机会稍纵即逝
✅ **低风险** - 不赌博，只做有把握的

| 特征 | 套利 | 赌博 |

| 需要信息 | 准确、实时 | 不需要 |

## 第四步：风险控制

### 止损原则

**1. 硬止损**
设置自动止损点（如亏损5%立即卖出）

**2. 软止损**
时间止损（如24小时未成交则放弃）

**3. 单笔止损**
单笔最大损失不超过总资金5%

**4. 总止损**
当日总亏损超过10%则停止操作

### 资金管理

```
单笔投入 = 总资金 × 风险系数（建议≤20%）
分散投入 = 分到3-5个不同套利机会
应急资金 = 保留20%作为备用金
```

### 实际案例

```
总资金：¥10,000

单笔投入：¥2,000（20%）
分散到3个机会：
- 机会1：¥600
- 机会2：¥700
- 机会3：¥700
应急资金：¥2,000

风险控制：如果某机会亏损20%，立即止损
最大亏损：¥140（0.7%总资金）
```
